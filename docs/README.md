# Cardmatch

Next.js 全端專案（App Router、API Routes、Prisma + PostgreSQL、iron-session）。

歡迎使用 Cardmatch 文件中心。在左側選單中可以找到：

- **開始使用**：本地環境設定、安裝、Docker 開發環境
- **功能說明**：約戰流程、即時同步機制、多帳號測試
- **API 文件**：REST API 規格與 OpenAPI 互動文件
- **測試**：單元、整合與 E2E 測試指南
- **部署**：部署到 Vercel 的完整流程
